01
安裝

如何下載並安裝 App？

請前往下載頁面取得最新版本。我們提供兩種安裝方式：

1

快速安裝（推薦）

打開 Terminal，貼上以下指令即可一鍵完成：

$ curl -L https://github.com/cashwu/WhispifyApp/releases/latest/download/Whispify.zip -o ~/Downloads/Whispify.zip && unzip -o ~/Downloads/Whispify.zip -d /Applications && xattr -cr /Applications/Whispify.app && open /Applications/Whispify.app
2

手動安裝

從 GitHub 下載 .dmg 檔案，安裝後需執行解除隔離指令。詳細步驟請參考下載頁面。

為什麼需要解除隔離？
由於 App 尚未經過 Apple 公證，macOS 的 Gatekeeper 會阻止開啟。使用快速安裝指令會自動處理。

02
方案

免費版和專業版有什麼差異？

免費版包含：

  • Whisper 模型：Tiny ~ Medium
  • 最多 3 個排程任務
  • 基本字幕匯出（SRT）

專業版額外解鎖：

  • 全部 Whisper 模型（含 Large V2、V3、V3 Turbo）
  • 無限排程任務
  • GPU 加速（速度提升 2-10 倍）
  • 雙語字幕匯出（原文/譯文/雙語）
  • 可用於 3 台設備

詳細比較請參考 價格方案 頁面。

03
授權

如何啟用與管理 Pro 授權？

購買 Pro 版本後，您將收到一組序號。請按照以下步驟啟用：

啟用 Pro 授權

1

開啟 Whispify Pro，點選 管理授權

2

點選 輸入序號

3

輸入您的序號，點擊驗證按鈕完成啟用

提醒：每組序號最多可在 3 台設備上使用。

解綁授權

若您需要更換裝置或解除綁定：

1

開啟管理授權，您會看到：解綁序號 和 清除所有綁定裝置

  • 解綁序號：僅解除當前裝置的綁定
  • 清除所有綁定裝置：解除此序號的所有裝置綁定

支援

如何回報問題或建議功能？

05
功能

App 支援哪些語言？

Whispify 使用 OpenAI 的 Whisper 模型進行語音辨識，目前支援：

  • 英文（English）
  • 日文（日本語）
  • 中文

您可以手動指定來源語言以提高辨識準確度。未來版本將加入更多語言支援。

06
隱私

我的影片會被上傳到雲端嗎？

不會！您的影片和音訊檔案完全不會離開您的裝置。

Whispify 使用 WhisperKit（基於 Apple CoreML）在您的 Mac 本機執行語音辨識：

  • 所有語音轉文字處理都在本機完成
  • 無需網路連線即可進行語音辨識
  • 處理完成後，原始檔案保持不變

關於 AI 翻譯功能：

當您使用 AI 翻譯功能時，僅會將字幕文字（非影片檔案）傳送至您所設定的 AI API 服務進行翻譯。

詳細資訊請參閱我們的 隱私政策。
