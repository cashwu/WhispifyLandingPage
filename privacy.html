隱私政策

最後更新日期：2025 年 12 月

Whispify 致力於保護您的隱私。本隱私政策說明我們的應用程式如何處理您的資料，以及我們採取的隱私保護措施。

核心原則：Whispify 的語音辨識功能完全在您的 Mac 本機執行，您的影片和音訊檔案不會上傳至任何伺服器。

1. 資料收集

Whispify 在大多數情況下不會收集您的個人資料。我們不使用任何分析工具或追蹤技術。

我們不收集的資料：

  • 個人身份資訊（姓名、電子郵件等）
  • 您的影片或音訊檔案
  • 使用行為或分析數據

專業版授權驗證：

當您啟用 Whispify 專業版時，為了驗證授權有效性，我們會收集並傳送以下資料至 Whispify 伺服器：

  • 您的 Mac 硬體識別碼（Hardware UUID）
  • 您輸入的產品序號

這些資料僅用於驗證序號有效性及確認授權設備數量，我們不會將其用於其他目的或與第三方分享。

2. 本機語音辨識

Whispify 使用 WhisperKit（基於 Apple CoreML）在您的 Mac 本機執行語音辨識。這意味著：

  • 您的影片和音訊檔案完全不會離開您的裝置
  • 所有語音轉文字處理都在本機完成
  • 無需網路連線即可進行語音辨識
  • 處理完成後，原始檔案保持不變

3. AI 翻譯功能

當您使用 AI 翻譯功能時，字幕文字會傳送至您所設定的 AI API 服務進行翻譯。請注意：

  • 僅傳送文字字幕，不包含影片或音訊
  • 您可以自行選擇和設定 API 服務提供商
  • API 金鑰以加密方式安全地儲存在應用程式的本機資料庫中
  • 翻譯請求直接傳送至您設定的 API 端點

重要提示：使用第三方 API 服務時，請參閱該服務提供商的隱私政策，了解其如何處理傳送的文字資料。

4. 資料儲存

Whispify 在您的本機裝置上儲存以下資料：

  • 應用程式設定：您的偏好設定和 API 配置
  • Whisper 模型：下載的語音辨識模型檔案
  • 輸出字幕：產生的 SRT 字幕檔案（儲存於您指定的位置）
  • 序號與 API 金鑰：以加密方式儲存在應用程式資料庫中

所有資料都儲存在您的 Mac 上，您可以隨時刪除這些資料。

5. 第三方服務

Whispify 可能與以下類型的第三方服務互動：

  • AI API 服務：用於字幕翻譯（僅當您啟用翻譯功能時，傳送至您自行設定的 API 端點）
  • 模型下載服務：用於下載 Whisper 語音辨識模型檔案

您所選擇的第三方服務各有其隱私政策，建議您在使用前閱讀了解。

6. 資料安全

由於 Whispify 採用本機處理架構，您的資料安全主要取決於：

  • 您的 Mac 裝置安全性
  • macOS 內建的安全保護機制
  • 您選擇的 API 服務提供商的安全措施

7. 兒童隱私

Whispify 並非專門針對兒童設計，我們不會刻意收集任何使用者（包括兒童）的個人資料。

8. 政策更新

我們可能會不定期更新本隱私政策。任何重大變更將透過應用程式更新或官方網站公告通知使用者。
