下載 Whispify

目前版本 v1.0.1 Beta · 支援 macOS 14.0+

macOS 快速安裝（推薦）

使用 Terminal 一鍵下載並安裝

打開 Terminal，貼上以下指令即可自動下載、解除隔離並開啟安裝程式：

curl -L https://github.com/cashwu/WhispifyApp/releases/download/v1.0.1/Whispify-v1.0.1.zip -o ~/Downloads/Whispify.zip && unzip -o ~/Downloads/Whispify.zip -d /Applications && xattr -cr /Applications/Whispify.app && open /Applications/Whispify.app

此指令會下載 .zip 檔案，自動解壓縮到 Applications 資料夾，移除 macOS 隔離標記，然後開啟應用程式。

macOS 手動下載

支援 macOS 14.0 (Sonoma) 或更新版本

1

下載安裝檔

前往 GitHub Releases 頁面下載 Whispify-v1.0.1.dmg 檔案。

前往 GitHub Releases
2

安裝應用程式

雙擊下載的 .dmg 檔案解壓縮，將 Whispify.app 拖曳到 Applications 資料夾。

3

解除 macOS 隔離（重要）

由於應用程式未經 Apple 公證，macOS 會阻止開啟。請在 Terminal 執行以下指令解除隔離：

xattr -cr /Applications/Whispify.app

這個步驟只需要執行一次，之後就可以正常開啟應用程式了。

4

開啟 Whispify

在 Applications 資料夾中找到 Whispify 並開啟，開始使用！

為什麼需要解除隔離？

macOS 的 Gatekeeper 安全機制會對從網路下載的未公證應用程式添加「隔離」標記。這是為了保護使用者免受惡意軟體的侵害。由於 Whispify 目前處於 Beta 階段，尚未申請 Apple 開發者帳號進行公證，因此需要手動解除隔離才能開啟。

自動更新功能

好消息！安裝完成後，Whispify 會自動檢查更新。當有新版本時，應用程式會提示您下載並安裝，無需再執行隔離解除步驟。您只需要完成一次手動安裝，之後的更新都會自動處理。
