Download

取得 Whispify

本地端影片字幕轉錄與翻譯工具

macOS 14.0+
Terminal — zsh
$ curl -L https://github.com/cashwu/WhispifyApp/releases/latest/download/Whispify.zip -o ~/Downloads/Whispify.zip && unzip -o ~/Downloads/Whispify.zip -d /Applications && xattr -cr /Applications/Whispify.app && open /Applications/Whispify.app

此指令會自動下載、解壓縮、移除隔離標記並開啟應用程式

1

下載安裝檔

前往 GitHub Releases 頁面下載最新的 .dmg 檔案

GitHub Releases →
2

安裝應用程式

雙擊 .dmg 檔案，將 Whispify.app 拖曳到 Applications 資料夾

3

解除 macOS 隔離

由於應用程式未經 Apple 公證，需在 Terminal 執行以下指令：

Terminal
$ xattr -cr /Applications/Whispify.app

此步驟只需執行一次

4

開始使用

在 Applications 資料夾或 Launchpad 中找到 Whispify 並開啟
