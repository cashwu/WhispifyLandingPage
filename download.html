本地端影片字幕轉錄與翻譯工具

Apple Silicon · macOS 14.0+
Windows 10+ (64-bit)
Terminal — zsh
$ curl -L https://github.com/cashwu/WhispifyApp/releases/download/1.1.1/Whispify.zip -o ~/Downloads/Whispify.zip && unzip -o ~/Downloads/Whispify.zip -d /Applications && xattr -cr /Applications/Whispify.app && open /Applications/Whispify.app

此指令會自動下載、解壓縮、移除隔離標記並開啟應用程式
