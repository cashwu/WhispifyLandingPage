v1.0.0-beta-1

Beta 🎉 首次發布
2025 年 12 月 21 日 macOS

🚀 Whispify 首個公開測試版本正式發布！這是一款 macOS 本地的影片字幕轉錄與翻譯工具。

新功能
  • 本地語音轉文字 — 使用 WhisperKit 在本機執行，完全保護隱私
  • 多語言辨識 — 支援英文、日文、中文語音辨識
  • AI 翻譯整合 — 支援多種 AI API 服務進行字幕翻譯
  • 多種 Whisper 模型 — 支援 Tiny ~ Medium 模型
  • 字幕匯出 — 支援 SRT 格式匯出
  • 排程處理 — 支援最多 3 個排程任務
在 GitHub 查看完整 Release
